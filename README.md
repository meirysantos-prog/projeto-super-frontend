# projeto-super-frontend

Projeto acadêmico desenvolvido em equipe.(foi realizado em 2024)

## Atuação
- Front-end

## Descrição
O projeto teve como objetivo o desenvolvimento de um sistema de monitoramento.
Minha atuação foi voltada para a criação das interfaces e desenvolvimento
das telas da aplicação.

## Tecnologias e Ferramentas
- Figma (prototipação e design de interfaces)
- Python
- Tkinter

## Atividades Desenvolvidas
- Criação de protótipos de telas no Figma
- Desenvolvimento da interface gráfica da aplicação
- Implementação de telas utilizando Tkinter
- Trabalho em equipe
